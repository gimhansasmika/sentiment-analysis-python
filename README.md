# 🤖 Sentiment Analysis Project (Python)

A machine learning project that predicts sentiment (Positive, Negative, Neutral) from text input.

---

## 🚀 Features
- Analyze user input text
- Classify sentiment:
  - 😊 Positive
  - 😐 Neutral
  - 😠 Negative

---

## 🛠️ Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy

---

## 📊 Dataset
- Kaggle dataset used for training

---

## ▶️ How It Works
1. Train model using labeled dataset  
2. User inputs text  
3. Model predicts sentiment  

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
